В папке "src" распологаются 4 файла из которых:
  2 файла написаны на TypeScript и 2 файла скомпилированны в готовый JavaScript код.


Оба сервера используют фреймворк Fastify.

Server 1 (server1.ts):
Server 1 - HTTP-сервер, который работает на 3001 порту. Когда приходит GET-запрос на маршрут "/" (корневой), он делает запрос к Server 2 по адресу http://localhost:3002, используя библиотеку axios. Затем Server 1 обрабатывает ответ от Server 2 и отправляет его содержимое обратно в качестве ответа на исходный запрос к Server 1.

Если в процессе выполнения запроса к Server 2 возникает ошибка (например Server 2 не работает), то вместо ответа от Server 2, Server 1 отправляет JSON-объект с сообщением об ошибке.

Server 2 (server2.ts):
Server 2 - это еще один HTTP-сервер, который работает на порту 3002. У него есть только один маршрут "/" (корневой), который обрабатывает GET-запросы. Когда на него приходит GET-запрос, сервер формирует JSON-ответ с определенным сообщением и отправляет его обратно в качестве ответа на запрос.
